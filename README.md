# Dreaded Clawd

**Power in Every Trade.** — A premium decentralized trading network connecting verified vendors and buyers.

## Tech Stack

- **Next.js 15** — React framework
- **Tailwind CSS** — Styling
- **Framer Motion** — Animations
- **TypeScript** — Type safety

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
